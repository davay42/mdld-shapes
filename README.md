# MDLD-SHACL Catalog

This is a catalog of SHACL constraints and targeting mechanisms represented in MDLD (Markdown Linked Data) text format.

Start here:[Shapes language catalog](./index.md)

## View with Intelligraphs

Clone the repo via [Intelligraphs app](https://app.intelligraphs.com/#/?clone=https://github.com/davay42/mdld-shapes.git)

## Navigate with IG-CLI

```bash
mkdir mdld-shapes
cd mdld-shapes
git clone https://github.com/davay42/mdld-shapes.git 
curl -O https://app.intelligraphs.com/ig-cli.js
chmod +x ig-cli.js
mv ig-cli.js /usr/local/bin/ig-cli
ig-cli --help
ig-cli stats
```
